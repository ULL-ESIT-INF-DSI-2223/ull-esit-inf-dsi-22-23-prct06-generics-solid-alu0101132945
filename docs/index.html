Práctica 4 - Arrays, tuplas y enumerados

En esta práctica tendremos que resolver una serie de ejercicios de programación que nos permitirán conocer más en profundidad los arrays, tuplas y enumerados de TypeScript.

Todas las soluciones a los ejercicios deberán estar alojadas en el repositorio generado tras la aceptación de la asignación de GitHub Classroom. En ese sentido, utilice en dicho repositorio una estructura de proyecto similar a la que hemos visto en clase.

Además, tendrá que comentar en un informe la solución que ha diseñado para cada uno de los ejercicios propuestos.

Ejercicio 1 - El alergólogo

Escriba una función getAllergens que reciba una puntuación de alérgenos de una persona y que devuelva una lista con los alérgenos a los que la persona es alérgica. Los diferentes alérgenos deberán modelarse mediante un enumerado.

Por último, tenga en cuenta que la función podría recibir una puntuación que incluya alérgenos no contemplados en la lista, esto es, alérgenos cuya puntuación sea 256, 512, 1024, etc. Además, si el valor pasado como argumento no es entero y positivo, la función deberá retornar el valor undefined.

export function getAllergens(valaler : number){
   enum allegerns {  huevo = 1,
                     cacahuete = 2,
                     marisco = 4,
                     fresa = 8,
                     tomate = 16,
                     chocolate = 32,
                     polen = 64,
                     gato = 128};

   let pAllergens : string[];
   if(valaler >= 256){
      return undefined;
   }

   while(valaler > 0){
      if(valaler - allegerns.gato >= 0){
         valaler = valaler - allegerns.gato
         pAllergens.push("gato")
      }
      else if(valaler - allegerns.polen >= 0){
         valaler = valaler - allegerns.polen
         pAllergens.push("polen")
      }
      else if(valaler - allegerns.chocolate >= 0){
         valaler = valaler - allegerns.chocolate
         pAllergens.push("chocolate")
      }
      else if(valaler - allegerns.tomate >= 0){
         valaler = valaler - allegerns.tomate
         pAllergens.push("tomate")
      }
      else if(valaler - allegerns.fresa >= 0){
         valaler = valaler - allegerns.fresa
         pAllergens.push("fresa")
      }
      else if(valaler - allegerns.marisco >= 0){
         valaler = valaler - allegerns.marisco
         pAllergens.push("marisco")
      }
      else if(valaler - allegerns.cacahuete >= 0){
         valaler = valaler - allegerns.cacahuete
         pAllergens.push("cacahuete")
      }
      else if(valaler - allegerns.huevo >= 0){
         valaler = valaler - allegerns.huevo
         pAllergens.push("huevo")
      }
   }
   return pAllergens
}

En este ejercicio se añade los valores de los alergenos, despues de comprobar que el valor pasado es correcto, a un array que sera el resultado de los alergenos una vez alcanzado el 0 en el valor de los alergenos.

Ejercicio 2 - Números complejos

Con lo visto hasta ahora en la asignatura, defina un tipo de datos propio que permita representar un número complejo, esto es, pares de valores numéricos reales, donde la primera componente del par es la parte real del complejo, mientras que la segunda componente del par representa su parte imaginaria.

A continuación, partiendo de dicha definición, escriba funciones que permitan calcular las siguientes operaciones sobre números complejos:

Suma, resta, multiplicación y división (funciones add, sub, mult y div). Estas funciones reciben como argumentos dos complejos y devuelven un complejo. Producto escalar (función prod). Esta función recibe como argumentos un complejo y un número real, retornando un número complejo. Conjugado (función conj). Recibe como argumento un complejo y devuelve otro complejo. Módulo (función abs). La función recibe como argumento un complejo y retorna un valor real.

type Complex = {real:number,imag:number}

export function addComp(complex1:Complex,complex2:Complex){
    let complexresult : Complex  = {real:0, imag:0};
    complexresult.real = complex1.real + complex2.real;
    complexresult.imag = complex1.imag + complex2.imag;
    return complexresult;
} 

export function subComp(complex1:Complex,complex2:Complex){
    let complexresult : Complex = {real:0, imag:0};
    complexresult.real = complex1.real - complex2.real;
    complexresult.imag = complex1.imag - complex2.imag;
    return complexresult;
}

export function multComp(complex1:Complex,complex2:Complex){
    let complexresult : Complex  = {real:0, imag:0};
    complexresult.real = (complex1.real* complex2.real) - (complex1.imag * complex2.imag)
    complexresult.imag = (complex1.real * complex2.imag) + (complex2.real * complex1.real)
    return complexresult;
}

export function divComp(complex1:Complex,complex2:Complex){
    let complexresult : Complex  = {real:0, imag:0};
    complexresult.real = ((complex1.real * complex2.real) + (complex1.imag * complex2.imag)) / ((complex2.real * complex2.real) + (complex2.imag * complex2.imag));
    complexresult.imag = ((complex1.imag * complex2.real) - (complex1.real * complex2.imag)) / ((complex2.real * complex2.real) + (complex2.imag * complex2.imag))
    return complexresult;
}

export function prodComp(complex1:Complex, num1:number){
    let complexresult : Complex  = {real:0, imag:0};
    complexresult.real = complex1.real * num1;
    complexresult.imag = complex1.imag * num1;
    return complexresult;
}

export function conjComp(complex1:Complex){
    let complexresult : Complex  = {real:0, imag:0};
    complexresult.real = complex1.real;
    complexresult.imag = -(complex1.imag);
    return complexresult;
}

export function absComp(complex1:Complex){
    let result : number = Math.sqrt((complex1.real * complex1.real) + (complex1.imag * complex1.imag));
    return result;
}

En este ejercicio primero se crea el tipo de datos de numeros complejos, que es el tipo de datos para la representacion binomial, y despues de esto se usara para hacer las operaciones descritas en el ejercicio, siguiendo la formulacion binomial para cada caso, y devolviendo el resultado en el mismo formato.

Ejercicio 3 - No cabrees a la reina

Escriba una función checkAtack que, dada una estructura de datos como la anterior, devuelva un valor lógico indicando si ambas reinas podrían atacarse dadas las posiciones de las mismas. Tenga en cuenta que solo puede haber una reina blanca y una reina negra en el tablero. En caso de que lo anterior no suceda, la función deberá devolver el valor undefined.

Por último, el tablero debe consistir en, exactamente, 8 filas y 8 columnas, donde cada casilla puede contener alguno de los valores -, N o B, exclusivamente. Aunque la anterior comprobación podría llevarse a cabo a través del código fuente incluido en la función (en tiempo de ejecución), defina un tipo de datos adecuado que impida, desde el punto de vista del tipado (en tiempo de compilación), pasarle a la función checkAtack un tablero no válido, esto es, con un número de filas/columnas diferente a 8 y/o celdas con valores no válidos).

type chessBoard = ('-' | 'B' | 'N')[8][8];

export function checkAttack(board: chessBoard){
    let bPos : number[] = [];
    let nPos : number[] = [];
    for (let i = 0; i < 8; i++) {
        for (let j = 0; j < 8; j++) {
            if (board[i][j] === 'B') {
                bPos = [i, j];  
            }
            if (board[i][j] === 'N') {
                nPos = [i, j];  
            }
        }
    }
    if(bPos.length === 0 || nPos.length === 0){
        return undefined;
    } 

    let count = 0;
    while(count < 8){
        if(chessDia(bPos,nPos,count) || chessDia(nPos,bPos,count)) return true;
        else if(chessRow(bPos,nPos,count) || chessRow(nPos,bPos,count)) return true;
        else if(chessCol(bPos,nPos,count) || chessCol(nPos,bPos,count)) return true;
        count++;
    }
    return false;

}

function chessDia(pos1:number[],pos2:number[],value : number){
    if(pos1[0] + value === pos2[0] && pos1[1] + value === pos2[1]) return true;
    else return false;
}

function chessRow(pos1:number[],pos2:number[],value : number){
    if(pos1[1] + value === pos2[1] && pos1[0] === pos2[0]) return true;
    else return false;
}

function chessCol(pos1:number[],pos2:number[],value : number){
    if(pos1[0] + value === pos2[0] && pos1[1] === pos2[1]) return true;
    else return false;
}

En este ejercicio tenemos un tipo de dato que guarda un tablero de ajedrez, de tamaño 8x8 y que tiene definido los valores de las casillas vacias y de las dos reinas, que se pasara a la funcion principal que hasta alcanzar el tamaño maximo del tablero se pasa las posiciones de las reinas a las funciones que calcularan si estan en la misma diagonal, la misma vertical u horizontal.

Ejercicio 5 - Matrices espirales

Escriba una función getSpiralMatrix que, dado un entero positivo n representando el tamaño de una matriz cuadrada, devuelva una matriz (array bidimensional) con todos los números enteros en el rango [1, n*n] y que estén dispuestos en la matriz conformando una espiral. La espiral debe comenzar en la primera fila y columna de la matriz e irse completando siguiendo las agujas del reloj.

function getSpiralMatrix(n: number) {
    const matrix: number[][] = Array[n][n];
    let value = 1;
    let filaIni = 0;
    let filaFin = n - 1;
    let colIni = 0;
    let colFin = n - 1;
    while (filaIni <= filaFin && colIni <= colFin) {
        for (let j = colIni; j <= colFin; j++) {
        matrix[filaIni][j] = value++;
        }
        filaIni++;
        for (let j = filaIni; j <= filaFin; j++) {
        matrix[j][colFin] = value++;
        }
        colFin--;
        if (filaIni <= filaFin) {
        for (let j = colFin; j >= colIni; j--) {
            matrix[filaFin][j] = value++;
        }
        filaFin--;
        }
        if (colIni <= colFin) {
        for (let j = filaFin; j >= filaIni; j--) {
            matrix[j][colIni] = value++;
        }
        colIni++;
        }
    }
    return matrix;
}

en esta funcion crearemos una matriz por defecto que tendra el tamaño indicado por el valor que se le pasa por parametro, y luego iremos rellenando cada uno de las posiciones siguiendo las agujas del reloj aumentando un contador para los valores de la matriz.

Ejercicio 6 - Compresión de números en rangos

Escriba una función fromArrayToRanges que reciba un array o lista de números enteros y los comprima en rangos, es decir, que devuelva una cadena de caracteres con la compresión obtenida. Un rango, es decir, un conjunto de números consecutivos se representará mediante una cadena de caracteres con el primer y último número del rango separado por un guión bajo (_). Un rango de un único número será la cadena de caracteres que representa a ese ùnico número. Luego, una serie de rangos vendrá separada por comas (,). Escriba una función fromRangesToArray que lleve a cabo la operación inversa, es decir, que reciba como argumento una cadena de caracteres representando una serie de rangos y devuelva el array de números correspondiente.

export function fromArrayToRanges(nArray: number[]): string{
    let result: string = ""
    let auxn: number = 0
    nArray.forEach(number => {
        if(auxn == 0 && result == ""){
            auxn = number
            result += String(auxn)
        }
        else{
            if(auxn == number-1){
                auxn = number
                if (result[result.length-1] != "_") result += "_"
            }
            else{
                if(result[result.length-1] == "_" || result[result.length-1] == ","){
                    result += String(auxn)
                }
                result += ","
                auxn = number
            }
        }
    });
    return result
}

export function fromRangesToArray(ranges: string) {
    let aux = ranges.split(",");
    let aux2 = [];
    aux.forEach(element => {
        aux2.push(element.split('_'));
    });
    let aux3: number[] = [];
    aux2.forEach(element => {
        for(let i:number = parseInt(element[0],10); i <= parseInt(element[1],10); i++){
            aux3.push(i);
        }
    });
    return aux3;
}

en este ejercicio tenemos dos funciones que realizan operaciones opuestas, el primero calcula los rangos de valores que hay dentro de un array, calculando los rangos convirtiendolos en string y añadiendolos al resultado, hasta que termina de recorrer el array y devuelve el resultado. Mientras que la otra operacion, recibe una cadena y crea el array que contendra los valores, que despues rellenara calculando que valores estan dentro de los rangos descritos.

Ejercicio 7 - Decodificar resistencias

El objetivo de este ejercicio es crear un programa que nos ayude a calcular el valor de una resistencia sin tener que memorizar los valores de las bandas. Para ello, cree una función decodeResistor que recibe como parámetros los nombres de los colores de una resistencia como entrada y devuelve un número de dos dígitos indicando el valor de la resistencia. La función deberá devover un número de dos dígitos incluso si recibe más de dos colores como parámetros.

export function decodeResistor(band1: string,band2: string){
    enum resistance {negro,marron,rojo,naranja,amarillo,verde,azul,violeta,gris,blanco};
    let resistance1: number = resistance[band1.toLowerCase()];
    let resistance2: number = resistance[band2.toLowerCase()];
    let resultResist: string = String(resistance1) + String(resistance2);
    return Number(resultResist);
}

En este ejercicio se crea un enum con todas las resistencias ordenadas por el valor indicado y despues se busca los valores indicados por los parametros en el enum, que se concatenaran en el resultado para devolver el valor numerico de la resistencia indicada.

Ejercicio 8 - Palabras encadenadas en un array

Para resolver este ejercicio, escriba una función meshArray que compruebe si las cadenas del array están encadenadas o no. La función recibirá como parámetro un array de cadenas de texto y devolverá:

“Error al encadenar” si las cadenas del array no están encadenadas. Una cadena de texto que contenga las letras que encadenan las palabras del array. A priori no sabe cuantas letras encadenadas tendrán en común, pero al menos será una.

export function meshArray(wordsArray: String[]) {
  let coincidence: boolean;
  let posCoincidence: number;
  let result = '';
  let contCoincidence = 0;
  
  for (let j = 0; j < wordsArray.length - 1; j++) {
    coincidence = false;
    posCoincidence = 0;
    for (let i = wordsArray[j].length - 1; i >= 0; i--) {
      if (!coincidence && wordsArray[j][i] === wordsArray[j + 1][0]) {
        posCoincidence = wordsArray[j].slice(i, wordsArray[j].length).length;
        coincidence = true;
      }
    }
    if (coincidence && wordsArray[j].substr(-posCoincidence, wordsArray[2].length) === wordsArray[j + 1].slice(0, posCoincidence)) {
      result += wordsArray[j].substr(-posCoincidence, wordsArray[2].length);
      contCoincidence++;
    }
  }
  if (contCoincidence === (wordsArray.length - 1)) {
    console.log(result);
  } else {
    console.log('Error al encadenar');
  }
}

En este ejercicio se va recorriendo el array con las palabras para comprobar si estan encadenadas, y si el array no tiene valores se devuelve un error al encadenar. En esta funcion primero se comprueba si hay una subcadena final que se pueda encadenar con el inicio de la siguiente, si es asi se añade esta subcadena a la cadena resultado con los encadenados, que devolvera como resultado de la ejecucion de la funcion.
